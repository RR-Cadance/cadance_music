Privacy Policy

Updated 1 Sep 2025

Summary

We use Microsoft Clarity for user behavior insights to improve the site experience. We don't track you across the web for advertising purposes. The Cadance marketing site is static and self‑contained.

What we collect

  • Basic server logs from GitHub Pages (handled by GitHub).
  • User behavior data via Microsoft Clarity for site optimization.
  • Emails you send us at info.rondo@cadance.music.

What we do not collect

  • No advertising trackers or cross‑site tracking for marketing purposes.
  • No sale of personal data. Ever.

Email

If you email us, we'll use your message solely to respond and to provide support. We don't add you to mailing lists.

Microsoft Clarity

We use Microsoft Clarity to understand how visitors interact with our site. This helps us improve the user experience. Clarity collects anonymous usage data including page views, clicks, and scrolling behavior. This data is not used for advertising purposes. You can learn more about Clarity's privacy practices at Microsoft's Privacy Statement.

Third‑party services

Our site is hosted by GitHub Pages. Any limited logging performed by GitHub is subject to their policies. We also use Microsoft Clarity as described above.

Children

Cadance is not directed to children under 13. If you believe a child has provided personal information, contact us and we’ll delete it.

Changes

If we update this policy, we’ll post the new version on this page with an updated date.

Contact

Questions? Email info.rondo@cadance.music.
